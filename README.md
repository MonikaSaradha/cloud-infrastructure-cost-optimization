# Cloud Infrastructure Cost Optimization (AWS vs Azure)

## Overview

This project analyzes cloud infrastructure performance and cost efficiency by comparing Amazon Web Services (AWS) and Microsoft Azure.

It combines machine learning, cost modeling, and natural language processing (NLP) to recommend optimal cloud strategies for businesses across industries.

---

## Tools & Technologies

* Python (Google Colab)
* Machine Learning (Linear Regression, Random Forest)
* NLP (Topic Modeling, Text Analysis)
* Excel

---

## Problem Statement

Organizations often struggle to select the most cost-efficient and high-performing cloud platform.

This project aims to:

* Compare AWS and Azure performance
* Analyze cost structures
* Predict infrastructure costs
* Optimize resource allocation

---

## Analysis Performed

### 1. Performance Analysis

* Compared CPU, memory, and storage across AWS and Azure
* Identified differences in resource distribution

### 2. Cost Analysis

* Evaluated on-demand pricing models
* Identified cost drivers across instance types

### 3. Predictive Modeling

* Linear Regression for baseline prediction
* Random Forest for improved accuracy and non-linear modeling

### 4. Cost Optimization

* Built a model to recommend cost-efficient configurations

### 5. NLP & Text Analysis

* Analyzed industry articles (FMCG & Healthcare)
* Applied topic modeling (LDA)
* Identified major cloud usage trends

---

## Notebooks

* `Text_Analysis.ipynb` – NLP and topic modeling
* `TA.ipynb` – FMCG text analysis
* `TA_Healthcare_.ipynb` – healthcare analysis

---

## Key Findings

* AWS offers better cost efficiency
* Azure provides stronger memory configurations
* Random Forest improves prediction accuracy
* Multi-cloud strategy is recommended
